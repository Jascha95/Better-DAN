val maybe_tuppel : f:('a -> 'b -> 'c) ->
       fa:('a -> 'c) -> fb:('b -> 'c) -> 'a option -> 'b option -> 'c option
Runs the given function on the two arguments, if both are not None. There are the four cases: None,None -> None None,Some e -> Some (f2 e) Some e,None -> Some (f1 e) Some e1, Some e2 -> Some (f e1 e2)
