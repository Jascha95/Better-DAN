Module type GenVars.VAR_WITHOUT_IMG


module type VAR_WITHOUT_IMG = sig .. end
This module type is the signature of the typevars without the image operations. The type is used by the MapVar Module to modify the interface to the image. There the two functions allowing the modifications of the image are hidden.

The image of the vars are the from type img. Solving the constraints means maps variables to values in a way, that the constraints are satisfied.
