Subversion

Um Subversion benutzen zu können muss man sich für die Webdienste des Programmiersprachen Lehrstuhls registrieren. Bitte geben Sie als Berechtigungsgruppe progweb an. Um die Berechtigung zu bekommen, muss der Informatik-Account freigeschaltet sein. Das habe ich bereits gemacht, falls mir der Account-Name bekannt war. Falls nicht, senden Sie mir bitte eine E-Mail mit Ihrem Accountnamen und ich werde Sie umgehend freischalten.

Die URL des subversion-Repositoriums lautet: https://abacus.informatik.uni-freiburg.de/svn/proglang/sopra0405/gruppeX wobei X für die Nummer der jeweiligen Gruppe steht.

Es gibt auch einen Subversion Plugin für Eclipse. Um ihn zu benutzen, müssen die Versionen von Plugin, Eclipse und Subversion zueinander passen. Es ist also etwas tricky, aber es geht. Hier die Installation, wie sie Sebastian Mischke durchgeführt hat - Voraussetzung ist Eclipse 3.0:

  1. Falls nicht vorhanden subversion runterladen.
  2. Subclipse: In Eclipse unter Help -> search for new ... -> new remote site: "http://subclipse.tigris.org/update" und dann von dort das subclipse plugin runterladen.
  3. Jetzt kommt die Crux: Subclipse kann nicht mit dem SSL server umgehen, desshalb muss man ihn explicit mit dem subversion anmelden: "svn list https://abacus.informatik.uni-freiburg.de/svn/proglang/sopra0405/gruppeX" im subversion Verzeichnis ausführen und den key des SSL servers bestätigen.
  4. Jetzt lässt sich mit subclipse das Repository einrichten.
